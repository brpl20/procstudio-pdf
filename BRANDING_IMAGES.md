## Tips for Creating Icons

1. **Start with a high-res SVG** - Create your logo as SVG first
2. **Use a favicon generator** - Tools like [RealFaviconGenerator](https://realfavicongenerator.net/) can generate all sizes
3. **Keep it simple** - Icons need to be recognizable at 16x16 pixels
4. **Test on multiple backgrounds** - Ensure visibility on light/dark themes
